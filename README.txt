左侧用于显示图片，也可以把文件夹，单张或者多张图片拖进显示区域来导入图片，光标在显示区域内时可以用鼠标滚轮缩放，按住空格使用抓手工具。
支持的格式嘛，jpg，jpeg，png，webp，bmp，tif，tga，反正大部分图片格式都可以，导出倒是统一png格式。



右侧按从上到下的顺序说明。


选取颜色，可以用滑块或者输入框指定前景色的RGBA值并预览，四个通道的取值范围都是0~255，此颜色为抠图和填充的依据。背景色。。。。然而并没有背景色。。。


选中抠图在显示区域左击，会从点击的像素开始，若该像素的RGBA值与前景色之差均小于容差，将这个像素的颜色改为透明，深度优先遍历颜色之差小于容差的像素实现抠图。大概一秒只能处理万来个像素，挺慢的，抠图的过程会显示在显示区域。选中抠图在显示区域右击，将点击位置的像素颜色设置设置为前景色，再抠图，在清理（清理见下文）（这个大概是最常用的功能了）。

取色把前景色改为点击位置的像素的颜色。

涂色把离点击位置的距离小于笔刷大小一半的像素的颜色改为前景色。

填充和抠图差不多，也是深度优先遍历像素，但将颜色之差小于容差的像素的颜色改为前景色。

以上四个分别有快捷键1、2、3、4。


容差和笔刷大小都可以用滑块或者输入框设置，取值范围0~255。



选择目录将所选目录下的图片导入，不包括子目录，如果当前的图片已经编辑过，导入前会自动保存。

更换背景有棋盘格和纯黑两种可选，选择目录或者点下一张上一张后都会统计新图片的浅色像素还是深色像素多，分别自动设置背景是纯黑或者棋盘格。

上一张下一张选取导入的图片的上一张下一张，如果当前的图片已经编辑过，会自动保存；会自动选择背景；会清空备份记录。

全自动半自动会自动抠图，流程都是先扫描四周边缘的像素，选取出现次数最多的作为前景色，再裁剪，再依次以左上左下右上右下为起点抠图。半自动抠完当前的图片就停了，全自动会自动下一张，再接着抠，直到最后一张抠完。

有时候抠出来的图片会有一堆堆几个像素的色块因为颜色超过容差没被扣掉，清理就是用来对付这些小色块的。由于清理要遍历所有有颜色的像素，耗时很长。

二值化将图片里透明像素改成白色，不透明像素改成黑色，抠线稿的时候会用到。

裁剪会把图片主体（非前景色颜色的像素）四周的一圈前景色像素裁掉。

撤销重做就是字面意思，每次抠图、填充、涂色、裁剪都会自动备份。

保存会保存当前的图片到done目录下，文件名与源文件相同，done目录不存在会自动创建。
